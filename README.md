# MedIO
A blog Web Application For People Who Are Interested In Writing Blogs.  
Please Note As The App is in development stage i have taken blogs from web once the app is in production i will remove all the previous blogs here. 
# Technologies Used
 1 Node.JS + Express.JS  
 2 MongoDB  
 3 Bulma - CSS Framework  
 4 EJS  
 5 Stripe Api 
# Features
 1 Pagination   
 Blogs are paginated and per each page 6 blog results are displayed.  
 2 Authenticated   
 Blogs are only visible to other Users Only if they are authenticated.   
 3 Donations  
 Stripe Api is used to accept donations from the users.
