# MedIO
A blog Web Application For People Who Are Interested In Writing Blogs.
# Technologies Used
 1 Node.JS + Express.JS  
 2 MongoDB  
 3 Bulma - CSS Framework  
 4 EJS  
 5 Stripe Api 
# Features
 1 Pagination   
 Blogs are paginated and per each page 6 blog results are displayed.  
 2 Authenticated   
 Blogs are only visible to other Users Only if they are authenticated.   
 3 Donations  
 Stripe Api is used to accept contributions and donations from the users.
